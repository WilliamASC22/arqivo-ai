# Arqivo AI

Arqivo AI is a public multi-agent AI workflow assistant that helps users turn messy documents, notes, and case information into organized summaries, missing-information checks, risk flags, next-step plans, draft responses, and audit logs.

## Project Goal

The goal of Arqivo AI is to show how AI agents can support real-world operations workflows while keeping humans in control of final decisions.

## Planned Features

- Public demo page
- Multi-agent case/document analysis
- Summary Agent
- Intake Agent
- Missing Information Agent
- Risk Agent
- Planner Agent
- Message Drafting Agent
- Quality Review Agent
- Review Queue
- Audit Log
- Free-safe public deployment
- Optional local AI mode with Ollama

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Hosting: Vercel and Hugging Face Spaces
- AI: Rule-based agents for public demo, optional local LLMs for local demos
